---
title: 'Installation'
---

From the npm registry:

```sh
# Using npm
> npm install express-rate-limit
# Using yarn
> yarn add express-rate-limit
# Using pnpm
> pnpm add express-rate-limit
```

From Github Releases:

```sh
# Using npm
> npm install https://github.com/nfriedly/express-rate-limit/releases/download/v{version}/express-rate-limit.tgz
# Using yarn
> yarn add https://github.com/nfriedly/express-rate-limit/releases/download/v{version}/express-rate-limit.tgz
# Using pnpm
> pnpm add https://github.com/nfriedly/express-rate-limit/releases/download/v{version}/express-rate-limit.tgz
```

Replace `{version}` with the version of the package that you want to your, e.g.:
`6.1.0`.
